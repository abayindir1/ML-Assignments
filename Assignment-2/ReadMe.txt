Install the required packages: "numpy", "pandas", "mathplotlib", and "scikit-learn" is used in this code. Please install them by running following commands: "pip install numpy", "pip install pandas", "pip install scikit-learn", "pip install matplotlib".

After installing packages, you can run the python code and it print the table with required information and it will create another window that show the plot model history for each model with some information about each model. After you close that window, program will stop.

you can access to the original data set from the following link: https://archive.ics.uci.edu/dataset/53/iris

you can access to the csv version of the dataset, which I used, from the following link: https://github.com/abayindir1/ML-Assignments/blob/main/Assignment-2/iris.csv
